## 📥 INSTALLATION

### Step 1
Download the latest release.

### Step 2
Open Roblox folder:


Roblox
└── Versions
└── version-xxxxxxxxxxxx
└── PlatformContent
└── pc
└── textures


### Step 3
Copy files from the Dark Textures folder.

### Step 4
Paste them into:


PlatformContent\pc\textures


### Step 5
Click **Replace All**.

### Step 6
Restart Roblox.

🎉 DONE
